Pasos para subir a GitHub
1
Descarga los 7 archivos que te he generado arriba (favicon.svg, favicon-16x16.png, favicon-32x32.png, favicon-48x48.png, apple-touch-icon.png, favicon-512x512.png, favicon.ico)
2
Súbelos a la raíz de tu repo en GitHub → Add file → Upload files. Al mismo nivel que index.html, sobre-mi.html, proyectos.html y descargas.html.
3
Añade el bloque de código del <head> en tus 4 páginas HTML, justo después del <meta charset>. Dímelo y te meto el código yo directamente en todos los archivos.
4
Commit y push. GitHub Pages tarda ~1 minuto en actualizar. Si el browser hace caché, abre en incógnito o fuerza recarga con Ctrl+Shift+R.
